# Sistem Akademik SMA Tadika Mesra

Sistem manajemen akademik berbasis Java EE + Maven untuk kebutuhan SMA Tadika Mesra.

## Fitur

### 👤 Admin
- Manajemen Guru
- Manajemen Siswa
- Jadwal
- Laporan Akademik
- Pengumuman

### 👨‍🏫 Guru
- Lihat Jadwal
- Input Nilai
- Daftar Siswa
- Rekap Absensi

### 👩‍🎓 Siswa
- Lihat Nilai
- Jadwal
- Absensi
- Data Diri
- Administrasi

## Teknologi
- Java 21
- Jakarta EE / Servlet JSP
- Maven
- Apache Tomcat
- MySQL

## Setup
1. Clone repo
2. Import ke NetBeans
3. Sesuaikan `db.properties` (jika pakai)
4. Jalankan di Tomcat
